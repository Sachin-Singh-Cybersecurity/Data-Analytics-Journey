# Level 4 — Power BI Sales Dashboard

## Project Overview
This project focuses on creating an interactive sales dashboard using Power BI. The dashboard provides business insights related to sales, profit, orders, regions, and product categories.

The goal of this project was to learn:
- dashboard development
- KPI visualization
- interactive filtering
- business storytelling using data

---

# Tools Used
- Power BI
- Excel Dataset
- Data Visualization
- Business Analytics

---

# Dashboard Components

## KPI Cards
- Total Sales
- Total Profit
- Total Orders
- Average Sales

## Sales Analysis
- Sales by Region
- Sales by State

## Profit Analysis
- Profit by Category

## Interactive Filters
- Region slicer
- Category slicer

---

# Key Insights

- West region generated the highest sales.
- Technology category produced the highest profit.
- Central region showed comparatively weaker performance.
- Sales distribution varied significantly across states.
- Interactive filtering improved targeted business analysis.
